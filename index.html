Welcome

Digitally Exploring the World!

There are so many aspects of our world that are still unknown, underappreciated, or not fully understood. Today you have the ability to explore and uncover how our world works digitally. Digital Geographic Techniques like GIS, Remote Sensing (RS), GPS, Geostats, and Web Mapping are versatile with limitless applications. The only thing limiting you is developing your skillsets. Look below and learn how to digitally explore the world.

The Must Haves

  • Esri

    A suite of GIS software products, which operate on desktop, server, and mobile platforms.

  • GitHub

    Hosting for software development version control.

  • Brackets

    A source code editor with a primary focus on web development.

  • Leaflet

    An open-source JavaScript library for mobile-friendly interactive maps.

  • Font Awesome

    A toolkit for fonts and icons.

  • Bootstrap

    A free and open-source CSS framework directed at responsive, mobile-first front-end web development.

  • Adobe

    Multimedia and creativity software products.

  • Google Fonts

    Catalog of free and open source typography.

  • JSFiddle

    Online coding sandbox.

  • Python

    Easy to use programming language.

  • Node.js

    JavaScript run-time environment.

  • HTML

    The web's markup language.

  • Mapbox

    Tools for custom online maps and geospatial APIs.

  • Google Earth Engine

    Cloud-based geospatial processing platform.

  • EEFA Book

    Open access guide to learning Google Earth Engine.

  • QGIS

    Open source desktop GIS for geospatial analysis.

  • Leaflet GeoSearch

    Plugin for searching addresses and places in Leaflet maps.
